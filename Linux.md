1. Home directory

The homedirectory is where the user data for each specific user is created under the root directory. Its initial user directory will be created on that user name. It is represented by a ***`tilda (~)`*** symbol.

<pre>
<b><i>Output:</i></b>
  asthik@LM21:~$ pwd  
  /home/asthik
</pre>
